Weather Forecast App – Project Overview

The Weather Forecast App is a web-based application developed using HTML, CSS, and JavaScript that allows users to check real-time weather information for any city. The app fetches live weather data from a weather API and displays details such as temperature, humidity, wind speed, and weather conditions in a simple and attractive interface.

Objective

The main objective of this project is to provide users with accurate and instant weather updates through an easy-to-use application.

Features
.Search weather by city name
.Displays current temperature
.Shows humidity level
.Displays wind speed
.Weather condition icons
.Responsive and modern user interface
.Error handling for invalid city names

Technologies Used

.HTML – Structure of the webpage
.CSS – Styling and responsive design
.JavaScript – Functionality and API handling
.OpenWeatherMap API – Fetching live weather data

How the Project Works

.The user enters a city name in the search box.
.JavaScript sends a request to the weather API.
.The API returns weather information in JSON format.
.The application extracts the required data.
.Weather details are displayed on the screen dynamically.

Modules of the Project

1. User Interface Module

Handles the design of the application including:

Search bar
Weather display section
Icons and buttons

2. API Integration Module

Responsible for:

Sending API requests
Receiving weather data
Processing JSON responses
3. Weather Display Module

Displays:

Temperature
Humidity
Wind speed
Weather status
4. Error Handling Module

Shows error messages when:

Invalid city name is entered
API request fails
